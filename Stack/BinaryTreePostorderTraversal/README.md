# Binary Tree Postorder Traversal

---

## Problem
Return the postorder traversal of a binary tree.

Postorder traversal order: Left → Right → Root.

---

## Approach
We use recursion:
1. Traverse left subtree
2. Traverse right subtree
3. Add current node to result list

---

## Time Complexity
O(n)

## Space Complexity
O(n) recursion stack

---

## Language
Java
