# Sort Colors (Dutch National Flag Algorithm)

## Problem
Given an array containing only 0, 1, and 2, sort the array in-place.

---

## Approach
Use three pointers:
- `low` for placing 0s
- `mid` for traversal
- `high` for placing 2s

Traverse the array once and swap elements accordingly.

---

## Time Complexity
- **O(n)**

## Space Complexity
- **O(1)**

---

## Language
- Java
