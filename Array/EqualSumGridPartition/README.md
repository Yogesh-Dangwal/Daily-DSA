## Problem
Check if grid can be split by one horizontal or vertical cut such that both parts have equal sum.

---

## Approach
1 Calculate total sum.
2 Try horizontal cuts.
3 Try vertical cuts.
4 If any sum equals half → true.

---

## Time Complexity
O(m × n)

---

## Space Complexity
O(1)

---

## Example
Input:

[[1,4],
 [2,3]]

Output:

true

---

## Language
Java
